Github -
Creating a new branch - git checkout -b dev

# Link your local folder to GitHub
git remote add origin https://github.com/shailuk/data_engg.git

# Pull the existing files from GitHub
git pull origin main

# Create the 'dev' branch and switch to it
git checkout -b dev

#Stage, Commit and Push 
# Add all files (the .github folder and the flight_booking folder)
git add .

# Create the commit
git commit -m "feat: setup cicd with workload identity and project structure"

# Push to the dev branch on GitHub
git push origin dev
